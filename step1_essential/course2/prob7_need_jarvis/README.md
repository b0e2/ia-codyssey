# 문제7 자비스가 필요해!

마이크를 통해 음성을 녹음하고 체계적으로 파일을 관리할 수 있는 파이썬 기반의 오디오 기록 프로그램입니다.

## 🚀 주요 기능
- **음성 녹음:** 시스템 마이크를 인식하여 입력한 n초간 음성을 녹음합니다.
- **자동 저장:** 녹음된 파일은 `records` 폴더에 `YYYYMMDD-HHMMSS.wav` 형식으로 자동 저장됩니다.
- **기록 검색:** 시작 날짜와 종료 날짜(YYYYMMDD)를 입력하여 해당 기간 동안 기록된 녹음 파일 목록을 조회할 수 있습니다.

## 🛠️ 개발 환경 및 요구사항
- Python 3.x
- 패키지 매니저: `uv` 권장
- 외부 라이브러리: `pyaudio`
